# 자동 새로고침(60초) 작동 여부 안내

## 현재 코드 동작
- 코드상으로는 60초마다 자동으로 시세 데이터를 가져오도록(fetch) 구현되어 있음.
- 하지만 실제 작동 여부는 100% 보장되지 않음.

## 문제 가능성
- 네이버 시세 API가 GitHub Pages 같은 외부 도메인에서의 요청을 CORS 정책으로 차단할 수 있음.
- 차단되면 화면에 "자동조회 실패 — 직접 입력해주세요"라는 메시지가 표시됨.

## 확인 방법
1. GitHub Pages에 배포 후 페이지 접속
2. 브라우저 개발자도구 열기 (F12)
3. Console 탭 확인
4. "blocked by CORS policy" 관련 에러 메시지가 있는지 확인

## 막혔을 경우 해결 방법

### 방법 1: GitHub Actions로 주기적 갱신
- GitHub Actions가 1분~몇 분 주기로 서버 쪽에서 시세를 가져옴
- 가져온 데이터를 JSON 파일로 저장
- 웹페이지는 이 JSON 파일만 읽도록 수정
- 장점: CORS 문제 없음
- 단점: GitHub Actions 최소 실행 주기에 한계 있음

### 방법 2: CORS 프록시 서버 사용
- Cloudflare Workers 같은 무료 서비스로 프록시 서버 구축
- 프록시를 거쳐서 네이버 API 호출
