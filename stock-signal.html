    참고
    · 가격은 60초마다 자동 조회합니다. KIS API 설정 버튼에서 한국투자 App Key·Secret을 입력하면 실시간 시세를 사용합니다. 미설정 또는 CORS 차단 시 네이버 금융 API로 자동 전환되며, 둘 다 실패하면 현재가 / 전일가를 직접 입력해주세요.
    · 매수 적용 시 수량·평균단가가 자동 계산됩니다.
    · 매도 적용 시 수량은 차감되고 직전 매도 수익현황이 카드에 표시됩니다.
    · 종목 이름을 클릭하면 직접 수정할 수 있습니다.
    · 차트는 가격이 조회될수록 데이터가 누적됩니다 (일별: 당일, 주별: 최근 30일).
    · 종목 정보는 이 브라우저에만 저장됩니다.
