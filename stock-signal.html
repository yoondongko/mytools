내 포지션 신호등

매입평균단가 대비 / 전일가 대비 등락률을 기준으로 매매 신호를 알려줍니다.

대기 중
매우 손절 추천 손절 추천 관망 물타기 추천 버티기 매도 수익화 가능 상승장 매우상승장
아직 등록된 종목이 없습니다. 위에서 종목코드, 종목명, 매입평균단가를 입력해 추가하세요.
참고
· 가격은 60초마다 자동 조회를 시도합니다. 브라우저 환경의 CORS 정책상 외부 시세 API 접속이 차단될 수 있으며, 이 경우 상단 상태가 "자동조회 실패 — 수동 입력 필요"로 표시됩니다.
· 자동조회가 실패하면 카드의 현재가 / 전일가 칸을 직접 클릭해 값을 입력하세요. 수동 입력 값으로 신호가 즉시 갱신됩니다.
· 종목 정보(코드/이름/매입단가)는 이 브라우저에만 저장됩니다.
