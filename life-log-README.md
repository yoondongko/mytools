# 라이프 로그 (life-log.html)

개인 일상 추적 툴. 수면, 약 복용, 시간 사용, 할일을 기록하고 그래프로 확인.

---

## 데이터 저장

- **localStorage** (`lifeLog_v1` 키) — 같은 기기/브라우저 안에서만 유지
- 브라우저 데이터 초기화 시 삭제됨
- 기기 간 이동은 내보내기/불러오기(JSON) 사용

---

## 기능

### 접근 보호
- index.html에서 라이프 로그 클릭 시 비밀번호 요구 (life-log.html 인라인 처리)
- 직접 URL 접근 시에도 페이지 자체 오버레이로 차단
- 같은 세션(탭) 안에서는 한 번 인증하면 재입력 불필요
- 탭 닫으면 세션 초기화
- 비밀번호: 소스코드에 평문 저장 — 개인 툴 수준의 보호

### 오늘 요약
- 페이지 상단에 오늘 날짜·요일·현재 시각 표시 (30초마다 갱신)
- 취침 시각 / 기상 시각 / 수면 시간 / 복약 횟수 / 오늘 완료한 할일 수 표시

### 약 기록
- 카드 상단에 기록 시각(날짜/시각, 5분 단위) 입력 — "지금으로" 버튼으로 리셋
- 각 약 버튼 색상: 약별로 지정한 색상 반영
- **항목 수정** 버튼 → 모달에서 약 이름 추가/삭제, 색상 변경
- 약 버튼 클릭 시 입력된 시각으로 복약 기록

### 할일
- **항목 수정** 버튼 → 모달에서 할일 추가/삭제
- 빠른 추가: 카드 상단 입력칸 + Enter 또는 추가 버튼
- ⬜ 클릭 → 완료 시각 입력 모달 → 확인 시 ✅로 전환 + 완료 로그 기록
- ✅ 클릭 → 미완료로 되돌리기 (완료 로그는 유지)

### 시간 기록
시작 시각 / 종료 시각(5분 단위) / 내용(메모) 입력 후 카테고리 버튼 클릭으로 저장.

- 종료 날짜·시각은 시작보다 같거나 이후여야 함 (입력·수정 모두 검사)
- **😴 취침** 버튼: 시작~종료를 `sleep_start` + `sleep_end` 로그로 기록

| 카테고리 | 색상 |
|---|---|
| 📚 공부 | 연두색 `#a8d85a` |
| 🌱 자기계발 | 개나리노랑 `#f5d020` |
| 🏠 생활 | 밝은블루 `#5bb3f0` |
| 👨‍👩‍👧 가족 | 연보라 `#c5a8e8` |
| 💼 취업준비 | 핑크 `#f472b6` |
| ⭐ 기타 | 스카이블루그레이 `#90abc0` |

### 그래프 (탭 + 기간 필터)

**기간 필터:** 주간(최근 7일) / 월간(이번달 일별) / 년간(월별 집계)

| 탭 | 내용 |
|---|---|
| 수면 시간 | 날짜별 수면 시간 막대. 년간은 월 평균 |
| 수면 타이밍 | 취침(🌙)/기상(☀️) 이모티콘 포인트 + 취침→기상 점선 연결 |
| 복약 타이밍 | 약별 개인 지정 색상으로 복용 시각 스캐터 |
| 시간 분석 | 카테고리별 누적 막대. 년간은 월별 합계 |
| 종합 리포트 | 평균 수면 + 14일 카테고리 합계 바 + 최근 7일 도넛 차트 |

### 최근 기록
- 카테고리 필터: 전체 / 취침 / 복약 / 카테고리별(공부·자기계발 등) / 할일
- 취침+기상 쌍은 하나의 행으로 표시: `😴 취침 22:30 → ☀️ 기상 06:30 (8.0h)`
- 제목 클릭으로 접기/펼치기
- 각 항목 **수정** → 타입별 모달 (복약: 약 이름+시각 / 수면: 취침·기상 시각 / 시간기록: 카테고리+시작·종료+내용 / 할일완료: 텍스트+시각)
- 수면 수정 시 기상 >= 취침 검사, 시간기록 수정 시 종료 >= 시작 검사
- 각 항목 개별 삭제 / 수면 행 삭제는 취침·기상 양쪽 동시 삭제
- "전체 삭제" 버튼 (확인 후 삭제)

### 데이터 내보내기 / 불러오기
- **내보내기**: `lifelog-YYYY-MM-DD.json` 파일로 다운로드
- **불러오기**: JSON 파일 선택 시 기존 데이터와 병합 (중복 제외, 날짜순 정렬)
- todos, medColors 필드도 함께 내보내기/불러오기 포함

## 로그 데이터 구조

```json
{
  "medNames": ["약 이름1", "약 이름2"],
  "medColors": { "약 이름1": "#7898ae" },
  "todos": [
    { "id": "타임스탬프", "text": "할일 내용", "done": false }
  ],
  "logs": [
    { "id": "타임스탬프", "type": "sleep_start", "ts": "ISO 날짜" },
    { "id": "타임스탬프", "type": "sleep_end",   "ts": "ISO 날짜" },
    { "id": "타임스탬프", "type": "med",          "name": "약 이름", "ts": "ISO 날짜" },
    { "id": "타임스탬프", "type": "activity",     "category": "공부", "ts": "시작 ISO", "endTs": "종료 ISO", "content": "메모" },
    { "id": "타임스탬프", "type": "todo",         "text": "할일 내용", "ts": "완료 ISO" }
  ]
}
```
