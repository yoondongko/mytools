# 라이프 로그 (life-log.html)

개인 일상 추적 툴. 수면, 약 복용, 시간 사용을 기록하고 그래프로 확인.

---

## 데이터 저장

- **localStorage** (`lifeLog_v1` 키) — 같은 기기/브라우저 안에서만 유지
- 브라우저 데이터 초기화 시 삭제됨
- 기기 간 이동은 내보내기/불러오기(JSON) 사용

---

## 기능

### 접근 보호
- index.html에서 라이프 로그 클릭 시 비밀번호 요구 (`auth.js` 처리)
- 직접 URL 접근 시에도 페이지 자체 오버레이로 차단
- 같은 세션(탭) 안에서는 한 번 인증하면 재입력 불필요
- 탭 닫으면 세션 초기화
- 비밀번호: 소스코드에 평문 저장 — 개인 툴 수준의 보호

### 오늘 요약
페이지 상단에 오늘 취침 시각 / 기상 시각 / 수면 시간 / 복약 횟수 표시.

### 기록 시각 설정
- 수면·약 기록 시 사용할 날짜/시각 입력 (기본값: 지금)
- "지금으로" 버튼으로 현재 시각으로 리셋
- 과거 시각 소급 입력 가능

### 수면 기록
- **😴 잠든다** / **☀️ 일어났다** 버튼 — 현재 상태에 따라 자동 전환
- 기록 시각 입력칸의 시각으로 저장

### 약 기록
- 약 이름 등록 후 버튼으로 빠르게 기록
- 약 버튼 위의 × 로 버튼 삭제 (복약 이력은 유지)
- "+ 추가" 버튼으로 새 약 이름 등록
- 기록 시각 입력칸의 시각으로 저장

### 시간 기록
시작 시각 / 종료 시각 / 내용(메모) 입력 후 카테고리 버튼 클릭으로 저장.

| 카테고리 | 색상 |
|---|---|
| 📚 공부 | 연두색 `#a8d85a` |
| 🌱 자기계발 | 개나리노랑 `#f5d020` |
| 🏠 생활 | 밝은블루 `#5bb3f0` |
| 👨‍👩‍👧 가족 | 연보라 `#c5a8e8` |
| 💼 취업준비 | 핑크 `#f472b6` |
| ⭐ 기타 | 스카이블루그레이 `#90abc0` |
| 😴 잠자는시간 | 네이비블루그레이 `#4e6880` |
| 💊 약섭취 | 블루그레이 `#7898ae` |

### 그래프 (탭 전환)

| 탭 | 내용 |
|---|---|
| 수면 시간 | 날짜별 수면 시간 막대 그래프 (14일) |
| 수면 타이밍 | 취침/기상 시각 스캐터 (14일) |
| 복약 타이밍 | 약별 복용 시각 스캐터 (14일) |
| 시간 분석 | 카테고리별 일별 누적 막대 그래프 (14일, 스택) |
| 종합 리포트 | 평균 수면 + 14일 카테고리 합계 바 + 최근 7일 도넛 차트 |

### 최근 기록
- 최근 30개 기록 목록
- 각 항목 개별 삭제 가능
- "전체 삭제" 버튼 (확인 후 삭제)
- 시간 기록은 `📚 공부 · 내용 (2.5h) / 6/14 09:00–11:30` 형식으로 표시

### 데이터 내보내기 / 불러오기
- **내보내기**: `lifelog-YYYY-MM-DD.json` 파일로 다운로드
- **불러오기**: JSON 파일 선택 시 기존 데이터와 병합 (중복 제외, 날짜순 정렬)

```json
{
  "medNames": ["약 이름1", "약 이름2"],
  "logs": [
    { "id": "타임스탬프", "type": "sleep_start", "ts": "ISO 날짜" },
    { "id": "타임스탬프", "type": "sleep_end",   "ts": "ISO 날짜" },
    { "id": "타임스탬프", "type": "med",          "name": "약 이름", "ts": "ISO 날짜" },
    { "id": "타임스탬프", "type": "activity",     "category": "공부", "ts": "시작 ISO", "endTs": "종료 ISO", "content": "메모" }
  ]
}
```
